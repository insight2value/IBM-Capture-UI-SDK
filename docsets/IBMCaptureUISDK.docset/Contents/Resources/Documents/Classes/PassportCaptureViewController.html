PassportCaptureViewController

open class PassportCaptureViewController : UIViewController

A UIViewController that presents a full screen CameraView to detect an MRZ, extract data from it and upon completion, capture the image that was used for the data extraction.

  • Output

    The result of the PassportCaptureViewController operation. It includes the extracted data and the image from where the data was extracted.
