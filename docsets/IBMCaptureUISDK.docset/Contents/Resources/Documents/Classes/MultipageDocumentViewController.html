  • acceptOrRejectAllImportedImages(count:completion:)

    This method handles the case where the user wants to import a PDF with more than one image. For now, the import overwrites everything that was captured previously and displays an accept/reject screen with just a label indicating the amount of images that are going to be imported. If the user accepts, continue forward as usual. If the user rejects, reset everything to as if the step was new.
